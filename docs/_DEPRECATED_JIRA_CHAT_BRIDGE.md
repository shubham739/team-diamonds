# [DEPRECATED] Jira Chat Bridge

> **DEPRECATED — This component is archived and no longer in active use.**

This component was a lightweight bridge service used during cross-team testing to connect an external chat client service (Team 9) to this team's Jira AI `/chat` endpoint. It polled the chat service for new user messages, forwarded them to the Jira AI service for processing, and posted the replies back to the chat channel. It was never intended for production use and has been archived.
